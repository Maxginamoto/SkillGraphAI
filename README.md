# Skill Strength Advisor

A simple Create React App project for assessing a student's engineering domain, CV text, and skillset. It suggests practical areas to improve and opens a small goal-focused chatbot after the assessment.

## Run

```bash
npm start
```


## Build

```bash
npm run build
```

## Notes

- Built with Create React App, not Vite.
- The assessment and chatbot are local rule-based logic, so no API key is required.
- PDF reading is intentionally basic. If a PDF does not extract enough text, paste the CV text into the textarea for a better result.
